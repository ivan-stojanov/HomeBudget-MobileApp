Adding Income to certain Account
Home -> Accounts -> Click on certain Account -> Add Income to this Account

Delete Income
Home -> Incomes -> Click on certain Income -> "Delete This Income" will delete all instances of the Income from the system, if there are more than one / "Delete" button from the list will delete one instance of the Income that is at the same line with the button, if there is a list of instances.

* User can define repeat cycle for the Income that he is entering, so that certain Income is added after chosen Repeat Period by itself.
* With deletion, amount for the Income is extracted from the Account balance.
