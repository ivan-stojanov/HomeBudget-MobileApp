Adding Expense
Home -> Expenses -> Add Expense

Adding Expense to certain Account
Home -> Accounts -> Click on certain Account -> Add Expense to this Account

Delete Expense
Home -> Expenses -> Click on certain Expense -> "Delete This Expense" will delete all instances of the Expense from the system, if there are more than one / "Delete" button from the list will delete one instance of the Expense that is at the same line with the button, if there is a list of instances.

* Paid Bills are count as Expenses.
* User can define repeat cycle for the Expense that he is entering, so that certain Expense is added after chosen Repeat Period by the system.
* With deletion, amount for the Expense is added from the Account balance.
